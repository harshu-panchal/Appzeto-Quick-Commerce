## Interaction States

### Icon Selector Modal

#### Default State
- All 20 icons displayed in a 6-column grid
- Icons have gray color and white background
- Border is light gray

#### Hover State
- Icon background changes to light indigo (indigo-50)
- Border changes to indigo (indigo-500)
- Icon color changes to indigo (indigo-600)
- Smooth transition animation

#### Selected State
- Icon background is light indigo (indigo-50)
- Border is dark indigo (indigo-600, 2px)
- Icon color is indigo (indigo-600)
- Persists until another icon is selected

#### Search Active
- Icons filter in real-time as user types
- No matches shows: "No icons found matching '{search term}'"
- Search is case-insensitive

### Add/Edit Modal

#### No Icon Selected
- Left circle shows sparkle icon (placeholder)
- Button text: "Select Icon"
- Background: gradient from indigo-50 to purple-50

#### Icon Selected
